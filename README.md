# Expense Tracker

A simple Python-based expense tracker to record, manage, and analyze daily expenses using CSV files and basic data handling.

## Overview

This project helps you track personal expenses by:
- Storing expense data in a CSV file
- Adding and managing expenses via Python scripts
- Analyzing spending patterns using a Jupyter Notebook

Designed to be lightweight, beginner-friendly, and easy to extend.

## Project Structure

Expense_tracker/
│
├── expense_tracker.py # Main expense tracking script
├── expense.py # Expense class and core logic
├── exp_New.py # Experimental / alternative implementation
├── expense_tracker.ipynb # Jupyter Notebook for analysis
├── expenses.csv # CSV file storing expense data
└── README.md # Project documentation


## Features

- Add, view, and manage expenses
- Store expenses persistently using CSV
- Categorize expenses
- Analyze expenses interactively using Jupyter Notebook
- Easy to modify and extend
